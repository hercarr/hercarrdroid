Contenido

  • Sesión # 1

    Construcción de una aplicación

    Analizaremos:
    Arquitectura de la plataforma Android
    Estructura de una aplicación Android
    Instalación de la aplicación en un dispositivo
    Crearemos nuestra aplicación utilizando / conociendo:
    Activity
    Fragment
    Intent
    AndroidManifest
    MaterialDesign
    Navigation Drawer

  • Sesión # 2

    Patrones de diseño

    Implementaremos algunos de los patros de diseño más comunes en las aplicaciones Android, tales como:
    MVP
    ViewPager
    RecyclerView
    ViewHolder
    CardView

  • Sesión # 3

    Consumo de un REST API

    Implementaremos el consumo de Web Services basados en REST para mostrar contenido dinámico, analizando las librerias más utilizadas por la comunidad de desarrolladores:
    AsynTask
    Volley
    Retrofit
    Así como analizaremos los conceptos:
    JSON
    PullToRefresh

  • Sesión # 4

    Intereacción

    Anexaremos a nuestra aplicacion caracteristicas que permitan al usuario tener un nivel mayor de interacción, conoceremos el uso de:
    Camara
    Share content
    Local Notifications
    Google Maps

Conocimiento

¿Qué aprenderemos en este curso?

Durante el transcurso de las 4 sesiones estaremos trabajando con herramientas / conceptos que se encuentran involucrados constantemente en el mundo de desarrollo de aplicaciones móviles Android.

  • Android Studio

    Utilizaremos el IDE oficial para el desarrollo de aplicaciones Android

  • Debug

    Realizaremos debug de nuestra aplicacion ejecutandola directamente en nuestro dispositivo

  • Material Design

    Diseñaremos nuestra aplicación siguendo los lineamientos que especifica Google

  • Design Patterns

    Implementaremos algunos de los patros de diseño más comunes en las aplicaciones publicadas en PlayStore
